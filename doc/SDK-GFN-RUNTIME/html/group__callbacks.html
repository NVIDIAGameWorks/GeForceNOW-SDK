GfnRuntimeError gfnRegisterExitCallback ( ExitCallbackSig  exitCallback,
void *  pUserContext 
)
Description
Register an application function to call when Geforce NOW needs to exit the game.
Environment
Cloud
Usage
Register an application function to call when Geforce NOW needs to exit the game.
Parameters
exitCallback[in]- Function pointer to application code to call when Geforce NOW needs to exit the game.
pUserContext[in]- Pointer to user context, which will be passed unmodified to the registered callback. Can be NULL.
Return values
gfnSuccess- On success, when running in a Geforce NOW environment
gfnDLLNotPresent- If callback could not be registered
gfnInvalidParameter- If exitCallback function pointer is NULL.
gfnCallWrongEnvironment- If callback could not be registered since this function was called outside of a cloud execution environment
